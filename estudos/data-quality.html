Qualidade de Dados

Data Quality

A qualidade de dados (data quality) é um dos fatores mais determinantes para o desempenho de sistemas de aprendizado de máquina. Modelos de IA aprendem padrões diretamente dos dados utilizados no treinamento. Se esses dados contêm erros, inconsistências ou viés (bias), o modelo tende a reproduzir essas falhas, comprometendo a confiabilidade dos resultados.

A presença de dados de baixa qualidade pode gerar respostas imprecisas, inconsistentes ou até alucinações (hallucinations). Como os sistemas não compreendem contexto da mesma forma que humanos, dependem fortemente da estrutura e da precisão dos dados fornecidos durante o treinamento.

A avaliação humana (human evaluation) é essencial para garantir que os dados sejam classificados corretamente e estejam alinhados às diretrizes (guidelines). Esse processo melhora a qualidade do dataset e contribui para modelos mais confiáveis, consistentes e adequados ao uso no mundo real.

Fonte: AWS Responsible AI Practices (2026); Introduction to Data Science — IE University (2026).
